Vacation

學習目標

利用不同的主題及競賽，引導學生學習及運用程式邏輯、程式撰寫、機械與程式整合運用，以銜接未來的機器人課程。
課程時數有 16 個級別，共 192 堂，288 小時的課程。
上課時間：09:00～16:00

注意事項

  1. 為提供更完善的學習品質，主辦單位將保留活動及場地調整的權利。
  2. 逢颱風、地震、豪雨等天災，依照當地政府規定停課與否，本單位恕不另行通知。
  3. 為保護個人隱私，課程中所拍攝之圖影片，僅作為課程及相關活動推廣之用，若您不同意請主動告知主辦單位。
  4. 六人以上開班，名額有限，報名請快！
  5. 報名繳費後，因故不克參加者，於開課日40天(含)前提出，全額退費；於開課日16天(含)前提出，退回90%學費；課程前4至15日，退回75%學費；課程前1至3日，退回50%學費；課程開始後未到則不退費。
  6. 可包班授課，有需要請電洽 03-3582030
立即報名
線上報名
報名表送出後，會有客服人員主動與報名者聯繫，報名者也可直接來電 (03)358-2030 ，或是加 LINE 線上客服 ( ID：3582030 ) 直接報名。
繳費
客服人員與報名者聯繫，確認報名課程、金額、日期等相關資訊後，請報名者在三日內轉帳至覓謎爾機器人教育中心的帳戶。（戶名：覓謎爾科技有限公司，遠東商銀－桃園大有分行，帳號：03800-10000-5643）
寄發繳費確認通知
報名者較費後，請將轉帳的金額、轉帳帳戶的末5碼，告知 LINE線上客服，或是將匯款憑證拍照傳給客服，我們進行資料確認及對帳後，會將確認結果告知報名者
寄發行前通知
開課的前一週，我們將用 Line 寄簡訊提醒您。
